# ⚖️ Landing Page Advocacia - Leonardo

Este é um projeto de uma landing page profissional desenvolvida para escritórios de advocacia que buscam uma presença online sólida, elegante e funcional.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização personalizada com foco em UX (User Experience) e design responsivo.
* **JavaScript:** Manipulação de DOM para efeitos no scroll e validação de formulário.
